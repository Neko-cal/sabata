# bambara-asr

## Overview

This repository contains scripts, configurations, and utilities for **fine-tuning Automatic Speech Recognition (ASR) models** for the **Bambara language** using **NVIDIA NeMo**. It contains the codes and configurations used to train [RobotsMali/soloni-114m-tdt-ctc](https://huggingface.co/RobotsMali/soloni-114m-tdt-ctc) and [RobotsMali/stt-bm-quartznet15x5](https://huggingface.co/RobotsMali/stt-bm-quartznet15x5).Â The fine-tuning process of each experiments is detailed in the [**Draft Technical Report V1**](https://wandb.ai/yacoudiarra-wl/bam-asr-nemo-training/reports/Draft-Technical-Report-V1--VmlldzoxMTIyOTMzOA)
